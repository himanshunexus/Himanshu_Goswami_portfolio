# Minimalist Django Portfolio

A high-end minimalist personal portfolio built with Django templates, Tailwind CSS v3, and small JavaScript interactions for reveal animations, modal handling, and dark mode.

## Quick Start

1. Create a Python 3.12+ virtual environment.
2. Install Python dependencies:
   `pip install -r requirements.txt`
3. Install front-end dependencies:
   `npm install`
4. Run migrations:
   `python manage.py migrate`
5. Build Tailwind CSS:
   `npm run build:css`
6. Start the development server:
   `python manage.py runserver`

## Development

- Rebuild styles on change with `npm run dev:css`
- Manage content from the Django admin after creating a superuser
